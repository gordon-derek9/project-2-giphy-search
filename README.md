# Project 02 – Website with a Search Engine (Giphy API)

**Name:** Derek Gordon

## Overview
This project is a single-page website that allows users to search for GIFs using the Giphy API. Users can enter a keyword, submit the search, and view relevant GIFs displayed in a responsive grid layout. The project demonstrates the use of HTML, CSS (Flexbox), JavaScript, and API integration.

## How to Use
1. Open the website in a browser.
2. Type a keyword (for example: cats, dogs, movies) into the search bar.
3. Click the **Search** button.
4. GIF results related to the search term will appear on the page.

## Features
- Search input with submit button
- Fetches data from the Giphy Search API
- Dynamically displays GIFs on the page
- Responsive layout for desktop and mobile
- Custom CSS grid using Flexbox

## Technologies Used
- HTML
- CSS (Flexbox)
- JavaScript
- Giphy API

## Ideas for Future Improvement
1. Add pagination or infinite scrolling to load more GIFs.
2. Allow users to filter results by rating or GIF type.
3. Add a favorites feature so users can save GIFs.
